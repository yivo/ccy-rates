## Installation

```
gem install bundler
cd ~ && git clone git@github.com:yivo/ccy-rates.git
cd ~/ccy-rates && bundle install
ln -s ~/ccy-rates/bin/ccy-rates /usr/local/bin/ccy-rates
```
